# 화면 목록 (Screen Inventory)

전체 **70+개** 화면/다이얼로그/알림. 각 화면의 진입 경로, 관련 패킷, 구현 우선순위, 정의 위치를 정리한다.

| 표기 | 의미 |
|------|------|
| `CUI` | `console_ui_design.md` |
| `DLG` | `dialogs_and_actions.md` |
| `HLP` | `help_and_guides.md` |
| `ERR` | `error_and_recovery.md` |
| `NTF` | `notification_patterns.md` |
