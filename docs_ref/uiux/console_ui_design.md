# 콘솔 UI 화면 설계

> **원칙**: 구분선 `=` 60자 (헤더), `-` 60자 (섹션 구분), 뒤로가기 항상 `0`, 오류 `[오류]`, 성공 `[성공]`

---

## 1. 초기 화면 (InitialMenu)

```
============================================================
    SMU Division of Computer Engineering
    Chat Service System
============================================================

  1. LOG IN
  2. JOIN MEMBER
  3. QUIT SERVICE

> Enter Command Number:
```

---

## 2. 로그인 화면

```
============================================================
    LOG IN
============================================================

> Enter your ID   : alice
> Enter Password  : ****

  [성공] 로그인 완료. 환영합니다, 홍길동 님!
```

**오류 케이스**:
```
  [오류] 존재하지 않는 ID입니다.          ← LOGIN_RES|1
  [오류] 비밀번호가 일치하지 않습니다.    ← LOGIN_RES|2
  [오류] 이미 다른 곳에서 접속 중입니다. ← LOGIN_RES|3 (P1)
```

비밀번호 마스킹: `getch()` 루프, 입력마다 `*` 출력, Enter(13)/Backspace(8) 처리.

## 3. 회원가입 화면

```
============================================================
    JOIN MEMBER
============================================================

> Enter new ID        (max 20, 금지: : ; | ): alice
> Enter new Password  (max 19, 금지: : ; | ): ****
> Enter Nickname      (max 20, Enter=ID 사용): 홍길동
> Enter Status Message(max 100, optional    ):

  [성공] 가입 완료! 로그인해주세요.
```

**오류 케이스**:
```
  [오류] 이미 사용 중인 ID입니다.     ← REGISTER_RES|2
  [오류] 사용 불가 문자가 포함됐습니다. (: ; | \n 금지)
  [오류] 최대 길이를 초과했습니다.
```

---

## 4. 메인 로비 (ShowMainMenu)

```
============================================================
    [MAIN LOBBY]  홍길동 [ONLINE]
    상태메시지: 안녕하세요!
============================================================

  1. 친구 목록
  2. 채팅방 목록 (그룹)
  3. 오픈채팅방 목록
  4. DM (다이렉트 메시지)
  5. 마이페이지
  6. 설정
  7. 로그아웃

> Enter Command Number:
```

**상단 상태 배지**:
- `[ONLINE]` → online_status=1
- `[BUSY ]` → online_status=2
- `[OFF  ]` → online_status=3 (invisible, 본인에게는 표시)

    [친구 목록]  총 4명 (온라인 2명)
============================================================

  온라인
  ------
  1. [ON ] 김철수  (kimcs)  | 오늘도 열심히!
  2. [바쁨] 박민준 (parkmj) | 회의 중

  오프라인
  --------
  3. [OFF] 이영희  (leeya)  | 마지막 접속: 2시간 전

------------------------------------------------------------
  a. 친구 추가       b. 친구 요청 목록
  c. 유저/닉네임 검색  d. 친구 삭제 / 차단

```
------------------------------------------------------------

  1. DM 보내기
  2. 채팅방 초대
  3. 프로필 보기
  4. 친구 삭제
  5. 차단
  0. 뒤로

> Enter Command:
```

## 6. 친구 추가 화면

```
============================================================
    [친구 추가]
============================================================

> 추가할 유저 ID 입력 (0=취소): bob

  [성공] 친구 요청을 전송했습니다.
```

**응답 케이스**:
```
  [성공] 친구 요청을 전송했습니다.     ← FRIEND_ADD_RES|0
  [오류] 존재하지 않는 ID입니다.       ← FRIEND_ADD_RES|1
  [오류] 차단된 사용자입니다.          ← FRIEND_ADD_RES|2
  [알림] 이미 친구 관계입니다.         ← FRIEND_ADD_RES|3
```

## 10. 채팅방 비밀번호 입력 화면

```
============================================================
    [채팅방 입장]  개발팀 [비밀번호 필요]
============================================================

> 비밀번호 입력 (0=취소): ****

  [성공] 입장합니다...
```

**오류 케이스**:
```
  [오류] 비밀번호가 틀렸습니다.         ← ROOM_JOIN_RES|0 (비번 불일치)
  [오류] 방 인원이 가득 찼습니다.       ← ROOM_JOIN_RES|0 (인원 초과)
  [오류] 존재하지 않는 채팅방입니다.    ← ROOM_JOIN_RES|0 (방 없음)
```

## 38. last_seen 표시 케이스 모음 (FR-A06)

친구 목록·프로필·마이페이지에서 마지막 접속 시간 표시 형식:

| 경과 시간 | 표시 |
|-----------|------|
| 60초 미만 | `방금 전` |
| 1분 ~ 59분 | `N분 전` (예: `15분 전`) |
| 1시간 ~ 23시간 | `N시간 전` (예: `3시간 전`) |
| 1일 ~ 6일 | `N일 전` (예: `2일 전`) |
| 7일 이상 | `YYYY-MM-DD` (예: `2026-04-20`) |
| 데이터 없음 | `알 수 없음` |
| 현재 접속 중 | 표시 안 함 (온라인 상태로 대체) |

```c
void format_last_seen(const char *last_seen, char *out, int len) {
    if (!last_seen || strlen(last_seen) == 0) {
        snprintf(out, len, "알 수 없음");
        return;
    }
    time_t last = parse_timestamp(last_seen);
    time_t now  = time(NULL);
    double diff = difftime(now, last);

    if      (diff < 60)         snprintf(out, len, "방금 전");
    else if (diff < 3600)       snprintf(out, len, "%d분 전", (int)(diff / 60));
    else if (diff < 86400)      snprintf(out, len, "%d시간 전", (int)(diff / 3600));
    else if (diff < 86400 * 7)  snprintf(out, len, "%d일 전", (int)(diff / 86400));
    else                        snprintf(out, len, "%.10s", last_seen);
}
```

## 41. 색상 출력 (Windows 콘솔 ANSI)

```c
#define COLOR_RESET    "\033[0m"
#define COLOR_RED      "\033[31m"
#define COLOR_GREEN    "\033[32m"
#define COLOR_YELLOW   "\033[33m"
#define COLOR_BLUE     "\033[34m"
#define COLOR_MAGENTA  "\033[35m"
#define COLOR_CYAN     "\033[36m"
#define COLOR_WHITE    "\033[37m"
#define COLOR_GRAY     "\033[90m"
#define COLOR_BOLD     "\033[1m"

/* 메시지 유형별 색상 */
/* 시스템 메시지 */
printf(COLOR_GRAY "--- %s ---" COLOR_RESET "\n", content);
/* 멘션 강조 */
printf(COLOR_BOLD COLOR_YELLOW "[멘션] %s: %s" COLOR_RESET "\n", nick, content);
/* 귓속말 */
printf(COLOR_MAGENTA "[귓속말] %s → 나: %s" COLOR_RESET "\n", nick, content);
/* 일반 메시지 — 설정 색상 적용 */
printf("[%s] %s%s" COLOR_RESET ": %s\n", ts, nick_color, nick, content);
/* /me 액션 */
printf(COLOR_CYAN "* %s %s" COLOR_RESET "\n", nick, content);
```

SetConsoleTextAttribute(h, FOREGROUND_GREEN | FOREGROUND_INTENSITY);

SetConsoleTextAttribute(h, FOREGROUND_RED | FOREGROUND_GREEN | FOREGROUND_BLUE);
