# 도움말 및 가이드 화면

콘솔 클라이언트의 도움말, 명령어 안내, 이모티콘 가이드 등을 정리한다. `/help` 명령어로 진입하며, 컨텍스트(메인/채팅방)에 따라 표시 내용이 달라진다.

---

## 1. 메인 로비 도움말 (`/help`)

```
============================================================
    [도움말]  메인 로비
============================================================

  메인 메뉴 단축키
  ────────────────
  1  → 친구 목록           4  → DM
  2  → 그룹 채팅방 목록    5  → 마이페이지
  3  → 오픈채팅방 목록     6  → 설정
  7  → 로그아웃           0  → 뒤로 (해당 화면)

  공통 단축키
  ────────────
  Ctrl+C    → 프로그램 즉시 종료
  Enter     → 입력 확인 / 빈 값 = 취소
  Backspace → 비밀번호 입력 시 한 글자 삭제

  유용한 팁
  ──────────
  • 알림이 갑자기 표시되면 친구나 메시지가 도착한 것입니다.
  • 채팅방 내부에서는 슬래시(/) 명령어로 다양한 기능 사용 가능합니다.
  • 설정 변경은 [설정] 메뉴에서 영구 저장됩니다.

> Enter 키를 눌러 닫기:
```

## 2. 채팅방 내부 도움말 (`/help`)

```
============================================================
    [채팅방 도움말]
============================================================

  메시지 작성
  ────────────
  일반 텍스트 입력 → 전송

  메시지 액션
  ────────────
  /del <msg_id>             내 메시지 삭제
  /edit <msg_id> <내용>     메시지 수정 (5분 이내, 본인)
  /reply <msg_id> <내용>    답장 (인용 표시)
  /pin <msg_id>             핀 메시지 고정 (방장/관리자)

  대화/멤버
  ────────────
  /w <닉네임> <내용>        귓속말 (1:1 비공개)
  /me <동작>                액션 메시지 (* 닉네임 동작)
  /members                  멤버 목록 보기
  /invite <id>              친구 초대 (또는 친구 목록에서 선택)
  /search <키워드>          현재 방 메시지 검색

  방 관리 (방장/관리자)
  ─────────────────────
  /notice <내용>            공지 등록
  /kick <id>                멤버 강퇴
  /grant <id>               공동 방장 부여 (방장 전용)
  /revoke <id>              공동 방장 해제 (방장 전용)

  오픈채팅 전용
  ──────────────
  /open_nick <닉네임>       오픈채팅 닉네임 변경

  나가기
  ────────
  /leave                    채팅방 나가기
  /help                     이 도움말 표시

------------------------------------------------------------
  특수 입력
  ───────────
  @닉네임                   해당 사용자 멘션 (강조 알림)
  :smile: :heart: :sad: 등  자동 이모티콘 변환
                            전체 목록은 /help emoticons

> Enter 키를 눌러 닫기:
```

## 3. 이모티콘 가이드 (`/help emoticons`) (FR-M06)

```
============================================================
    [이모티콘 가이드]
============================================================

  메시지에 아래 코드를 입력하면 자동 변환됩니다.

  감정 표현
  ──────────
  :smile:    →  (^_^)         웃음
  :laugh:    →  (^o^)         크게 웃음
  :wink:     →  (^_-)         윙크
  :shy:      →  (>_<)         쑥스러움
  :cool:     →  (B_B)         쿨함

  슬픔/놀람
  ──────────
  :sad:      →  (T_T)         슬픔
  :angry:    →  (-_-)         화남
  :shock:    →  (O_O)         놀람
  :sweat:    →  (^_^;)        식은땀

  기타
  ─────
  :heart:    →  <3            하트
  :lol:      →  (LoL)         빵 터짐
  :wave:     →  ( ^_^)/       손 흔들기

  사용 예
  ────────
  > 메시지 입력: 안녕하세요! :wave: 잘 부탁드립니다 :smile:
  결과:         안녕하세요! ( ^_^)/ 잘 부탁드립니다 (^_^)

> Enter 키를 눌러 닫기:
```

## 4. 설정 화면 도움말 (`?` 입력)

```
============================================================
    [설정 도움말]
============================================================

  설정 변경 흐름
  ───────────────
  1. 번호 입력 → 변경할 항목 선택
  2. 새 값 입력 또는 옵션 선택
  3. 0 입력 → 모든 변경사항 저장 후 종료

  각 설정 설명
  ────────────
  1. 메시지 색상      내가 보내는 메시지 텍스트 색상
                      (다른 유저에게는 영향 없음)

  2. 닉네임 색상      내 닉네임 표시 색상
                      (다른 유저에게는 영향 없음)

  3. 테마             콘솔 배경 색상 (dark/light)

  4. 타임스탬프 형식  메시지 시간 표시 형식
                      • HH:MM (기본)
                      • HH:MM:SS (초까지)
                      • MM-DD HH:MM (날짜 포함)

  5. 방해금지 모드    모든 알림 무음 (멘션 제외)

  6. 온라인 상태      친구에게 보이는 내 상태
                      • online: 일반 접속
                      • busy: 바쁨 표시
                      • invisible: 오프라인으로 위장

> Enter 키를 눌러 닫기:
```

## 5. 첫 로그인 환영 화면

회원가입 직후 또는 첫 로그인 시 1회 표시:

```
============================================================
    환영합니다, 홍길동 님!
============================================================

  ────────────────────────────────────────────────
  [Quick Start]
  ────────────────────────────────────────────────

  1. 친구 추가
     메인 로비 → 1.친구 목록 → a.친구 추가
     상대방 ID를 알면 바로 요청 가능

  2. 그룹 채팅방 만들기
     메인 로비 → 2.채팅방 → c.방 만들기
     방 이름 / 인원 / 비번 / 주제 입력

  3. 오픈채팅 참여
     메인 로비 → 3.오픈채팅 → 번호 선택
     누구나 자유롭게 참여 가능

  4. 도움이 필요하면
     채팅방 안에서: /help
     메인 로비에서: /help (지원 시)

  ────────────────────────────────────────────────
  💡 Tip: 채팅방에서 :smile: 같은 코드를 입력하면
        자동으로 이모티콘으로 변환됩니다!

> Enter 키를 눌러 시작:
```

이후 로그인부터는 표시되지 않음 (`user_settings.txt`에 `welcome_shown` 플래그 또는 첫 로그인 시점만 검사).

## 6. 명령어 자동완성 가이드 (입력 중)

`/`로 시작하는 명령어의 일부만 입력 후 Tab 또는 `?` 입력 시:

```
> 메시지 입력: /m
  [명령어 후보]
    /me <동작>         액션 메시지
    /members           멤버 목록 보기
  > 메시지 입력: /me

> 메시지 입력: /n
  [명령어 후보]
    /notice <내용>     공지 등록 (방장)
  > 메시지 입력: /notice
```

> **구현 노트**: 콘솔 환경에서 Tab 자동완성은 한정적이므로, 입력 시작 시 `?`로 후보 표시를 권장.

## 8. /help 라우팅 매핑

| 입력 | 표시 |
|------|------|
| `/help` (메인 로비) | 메인 로비 도움말 |
| `/help` (채팅방) | 채팅방 도움말 |
| `/help emoticons` | 이모티콘 가이드 |
| `/help commands` | 채팅방 명령어 전체 |
| `/help <명령어>` | 특정 명령어 상세 (예: `/help reply`) |
| `?` (설정 화면) | 설정 도움말 |
| `?` (입력 중) | 명령어 후보 표시 |

```c
void route_help(const char *args, int context) {
    if (args == NULL || strlen(args) == 0) {
        if (context == CTX_LOBBY)    show_lobby_help();
        else if (context == CTX_CHAT) show_chat_help();
        else                          show_general_help();
    }
    else if (strcmp(args, "emoticons") == 0)  show_emoticon_guide();
    else if (strcmp(args, "commands")  == 0)  show_command_list();
    else                                      show_command_detail(args);
}
```

## 9. 명령어 상세 도움말 예시 (`/help reply`)

```
============================================================
    [명령어 상세]  /reply
============================================================

  형식    : /reply <msg_id> <내용>
  예시    : /reply 42 네, 동의합니다!
  설명    : 특정 메시지에 대한 답장(인용)을 전송합니다.
            수신자 화면에는 인용 원문 미리보기가 표시됩니다.

  관련    : /me, /w, /edit
  우선순위: P3 (Phase 3 이후)

> Enter 키를 눌러 닫기:
```
