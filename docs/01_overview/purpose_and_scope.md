# 목적과 범위

## 1. 목적

대학 교직원을 위한 온라인 학생관리 시스템은 Windows PC에서 동작하는 C 언어 기반 콘솔 프로그램이다. 학생 관리자는 클라이언트에서 로그인한 뒤 학생정보 관리 기능을 사용한다.

서버는 학생정보 관리 초기화 작업을 수행하고, 클라이언트 요청을 소켓으로 수신하여 처리한 뒤 결과를 반환한다.

## 2. In-Scope

| ID | 범위 |
|----|------|
| SCOPE-01 | Windows 콘솔 기반 C 프로그램 |
| SCOPE-02 | 서버/클라이언트 프로그램 분리 |
| SCOPE-03 | 소켓 통신 기반 요청/응답 |
| SCOPE-04 | 로그인 |
| SCOPE-05 | 로그인된 사용자만 학생정보 관리 |
| SCOPE-06 | 로그인된 사용자만 신규 사용자 추가 |
| SCOPE-07 | 사용자 삭제 |
| SCOPE-08 | 학생 검색, 추가, 삭제, 수정 |
| SCOPE-09 | 학과명 기반 전체 학생 조회 |
| SCOPE-10 | `users.txt`, `SMU_Students.txt` 파일 입출력 |
| SCOPE-11 | 서버의 연결 리스트 기반 학생정보 관리 |
| SCOPE-12 | 매일 01:00 `time.h` 기반 학생정보 저장 |

## 3. Out-of-Scope

최신 `requirements.md` 에 별도 제외 범위는 명시되지 않았다. 따라서 본 문서에 없는 기능은 구현 범위에 추가하지 않는다.
