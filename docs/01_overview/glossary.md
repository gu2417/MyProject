# 용어집

| 용어 | 정의 |
|------|------|
| 학생 관리자 | 클라이언트 프로그램에 로그인한 뒤 학생정보를 관리하는 인가 사용자 |
| 서버 | 학생정보 초기화, 요청 처리, 파일 저장을 담당하는 프로그램 |
| 클라이언트 | 사용자 명령을 입력받아 서버에 요청을 보내고 결과를 출력하는 프로그램 |
| 학생정보 | 이름, 학번, 소속학과, 학년, GPA, 이메일 연락처, 모바일 폰 연락처 |
| 사용자 정보 | 사용자ID, 사용자암호, 사용자 소속부서, 사용자 이름 |
| 연결 리스트 | 서버가 학생정보를 관리하기 위해 사용하는 동적 자료구조 |
| 메시지 | 클라이언트와 서버가 소켓으로 주고받는 탭 구분 문자열 |
| PAYLOAD | 메시지 내 명령별 데이터 영역 |
