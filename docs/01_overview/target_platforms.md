# 타깃 플랫폼

## 1. 실행 환경

| 항목 | 기준 |
|------|------|
| 운영체제 | Windows |
| 언어 | C |
| UI | Console |
| 통신 | Socket |
| 저장 | Text File |

## 2. 런타임 의존성

| 의존성 | 사용 목적 |
|--------|-----------|
| C 표준 라이브러리 | 문자열 처리, 파일 입출력 |
| `time.h` | 매일 01:00 저장 시점 판단 |
| Windows Socket API | 서버/클라이언트 소켓 통신 |

## 3. 이식성 기준

본 프로젝트는 Windows PC 실행을 기준으로 한다. 다른 운영체제 지원은 최신 요구사항에 포함되지 않는다.
