# 프로젝트 개요

## 1. 프로젝트 정보

| 항목 | 내용 |
|------|------|
| 프로젝트명 | 온라인 학생관리 시스템 |
| 사용자 | 대학 교직원 |
| 기준 문서 | `requirements.md` |
| 언어 | C |
| 운영체제 | Windows |
| UI | 콘솔 |
| 구조 | 서버/클라이언트 |

## 2. 프로젝트 범위

본 프로젝트는 학생 관리자가 클라이언트 프로그램으로 로그인한 뒤 학생정보를 관리할 수 있도록 지원한다. 서버는 학생정보 관리 초기화 작업을 수행하고, 클라이언트 요청을 처리하며, 학생정보 관리 서비스를 제공한다.

## 3. 기술 결정

| 결정 | 내용 |
|------|------|
| 콘솔 프로그램 | Windows PC에서 동작하는 C 콘솔 프로그램으로 개발 |
| 서버/클라이언트 | 학생정보 관리 서버와 학생 관리자용 클라이언트 분리 |
| 소켓 통신 | 클라이언트 요청과 서버 결과를 소켓으로 송수신 |
| 파일 저장 | `users.txt`, `SMU_Students.txt`를 탭 구분 파일로 관리 |
| 연결 리스트 | 서버가 학생정보를 연결 리스트로 관리 |
| 시간 처리 | 매일 01:00 저장 판단에 `time.h` 사용 |

## 4. 주요 기능

- 로그인
- 신규 사용자 추가
- 사용자 삭제
- 학생 검색
- 학생 추가
- 학생 삭제
- 학생 정보 수정
- 학과명 기반 전체 학생 조회
- 매일 01:00 학생정보 저장
