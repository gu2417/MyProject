# 매일 01:00 학생정보 저장

## 1. 기능 개요

서버는 매일 01:00에 학생정보 저장을 수행한다. 저장 시 학생 추가, 수정, 삭제 데이터가 `SMU_Students.txt` 에 반영되어야 한다.

## 2. 요구사항

| ID | 기능 | 설명 |
|----|------|------|
| FR-SAVE-001 | 저장 시점 | 서버는 매일 01:00에 학생정보 저장을 수행한다. |
| FR-SAVE-002 | 데이터 반영 | 추가, 수정, 삭제 데이터를 `SMU_Students.txt` 에 반영한다. |
| FR-SAVE-003 | 파일 입출력 | 학생정보 저장은 파일 입출력으로 처리한다. |
| FR-SAVE-004 | 시간 처리 | 저장 시점 판단에는 `time.h` 를 사용한다. |

## 3. 구현 기준

- 서버 루프에서 현재 시간을 확인한다.
- 하루에 한 번만 01:00 저장이 실행되도록 마지막 저장 날짜를 관리한다.
- 저장 대상은 서버 메모리의 학생 연결 리스트이다.
