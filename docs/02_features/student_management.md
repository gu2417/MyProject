# 학생정보 관리 기능

## 1. 학생 검색

| ID | 기능 | 설명 |
|----|------|------|
| FR-STU-SEARCH-001 | 검색 기능 | 로그인된 사용자는 학생 검색 기능을 사용할 수 있다. |
| FR-STU-SEARCH-002 | 검색 조건 | 학생 검색 조건은 요구사항 미정이다. |
| FR-STU-SEARCH-003 | 서버 처리 | 서버는 연결 리스트 기반으로 학생 검색 요청을 처리한다. |
| FR-STU-SEARCH-004 | 결과 출력 | 클라이언트는 서버 검색 결과를 출력한다. |

## 2. 학생 추가

| ID | 기능 | 설명 |
|----|------|------|
| FR-STU-ADD-001 | 추가 기능 | 로그인된 사용자는 학생 추가 기능을 사용할 수 있다. |
| FR-STU-ADD-002 | 필드 | 이름, 학번, 소속학과, 학년, GPA, 이메일 연락처, 모바일 폰 연락처를 포함한다. |
| FR-STU-ADD-003 | 리스트 반영 | 서버는 신규 학생 정보를 연결 리스트에 추가한다. |
| FR-STU-ADD-004 | 파일 반영 | 추가된 학생 정보는 `SMU_Students.txt` 에 반영된다. |

## 3. 학생 삭제

| ID | 기능 | 설명 |
|----|------|------|
| FR-STU-DEL-001 | 삭제 기능 | 로그인된 사용자는 학생 삭제 기능을 사용할 수 있다. |
| FR-STU-DEL-002 | 리스트 반영 | 서버는 학생 삭제 요청을 연결 리스트에 반영한다. |
| FR-STU-DEL-003 | 파일 반영 | 삭제된 학생 정보는 `SMU_Students.txt` 에 반영된다. |
| FR-STU-DEL-004 | 식별 기준 | 삭제 대상 식별 기준은 요구사항 미정이다. |

## 4. 학생 정보 수정

| ID | 기능 | 설명 |
|----|------|------|
| FR-STU-UPD-001 | 수정 기능 | 로그인된 사용자는 학생 정보 수정 기능을 사용할 수 있다. |
| FR-STU-UPD-002 | 리스트 반영 | 서버는 학생 정보 수정 요청을 연결 리스트에 반영한다. |
| FR-STU-UPD-003 | 파일 반영 | 수정된 학생 정보는 `SMU_Students.txt` 에 반영된다. |
| FR-STU-UPD-004 | 수정 기준 | 수정 대상 식별 기준과 수정 가능 필드는 요구사항 미정이다. |
