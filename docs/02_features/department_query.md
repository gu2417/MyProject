# 학과명 기반 조회

## 1. 기능 개요

로그인된 사용자는 학과명을 기준으로 해당 학과의 전체 학생을 조회할 수 있다.

## 2. 요구사항

| ID | 기능 | 설명 |
|----|------|------|
| FR-DEPT-001 | 조회 기능 | 로그인된 사용자는 학과명 기반 전체 학생 조회 기능을 사용할 수 있다. |
| FR-DEPT-002 | 비교 기준 | 서버는 입력된 학과명과 학생의 소속학과를 비교한다. |
| FR-DEPT-003 | 결과 반환 | 서버는 해당 학과의 전체 학생 정보를 반환한다. |
| FR-DEPT-004 | 결과 출력 | 클라이언트는 서버 조회 결과를 출력한다. |

## 3. 구현 기준

- 서버는 학생 연결 리스트를 처음부터 끝까지 순회한다.
- `department` 필드가 입력 학과명과 일치하는 학생을 결과에 포함한다.
- 결과 메시지 포맷은 `08_api` 문서를 따른다.
