# 파일 입출력 정책

## 1. 로딩

| 파일 | 시점 | 처리 |
|------|------|------|
| `SMU_Students.txt` | 서버 시작 | 학생 연결 리스트 구성 |
| `users.txt` | 로그인/사용자 관리 | 사용자 정보 검증 및 변경 |

## 2. 저장

| 대상 | 시점 | 처리 |
|------|------|------|
| 학생정보 | 학생 추가/삭제/수정 반영 또는 매일 01:00 | `SMU_Students.txt` 갱신 |
| 사용자 정보 | 신규 사용자 추가/사용자 삭제 | `users.txt` 갱신 |

## 3. 실패 처리

파일 입출력 실패 시 서버는 실패 처리 후 클라이언트에 결과를 반환한다.
