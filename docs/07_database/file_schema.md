# 파일 스키마

## 1. `SMU_Students.txt`

```text
name<TAB>student_id<TAB>department<TAB>grade<TAB>gpa<TAB>email<TAB>mobile_phone
```

| 순서 | 필드 | 설명 |
|------|------|------|
| 1 | name | 이름 |
| 2 | student_id | 학번 |
| 3 | department | 소속학과 |
| 4 | grade | 학년 |
| 5 | gpa | GPA(평균학점) |
| 6 | email | 이메일 연락처 |
| 7 | mobile_phone | 모바일 폰 연락처 |

## 2. `users.txt`

```text
user_id<TAB>password<TAB>department<TAB>name
```

| 순서 | 필드 | 설명 |
|------|------|------|
| 1 | user_id | 사용자ID |
| 2 | password | 사용자암호 |
| 3 | department | 사용자 소속부서 |
| 4 | name | 사용자 이름 |
