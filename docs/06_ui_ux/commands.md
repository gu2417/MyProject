# 명령 목록

## 1. 로그인 전

| 명령 | 설명 |
|------|------|
| LOGIN | 사용자ID와 사용자암호로 로그인 |

## 2. 로그인 후

| 명령 | 설명 |
|------|------|
| STUDENT_SEARCH | 학생 검색 |
| STUDENT_ADD | 학생 추가 |
| STUDENT_DELETE | 학생 삭제 |
| STUDENT_UPDATE | 학생 정보 수정 |
| DEPT_LIST | 학과명 기반 전체 학생 조회 |
| USER_ADD | 신규 사용자 추가 |
| USER_DELETE | 사용자 삭제 |
