# 입력 검증

## 1. 학생정보 검증

| 필드 | 기준 |
|------|------|
| name | 빈 값 불가 |
| student_id | 빈 값 불가, 중복 불가 |
| department | 빈 값 불가 |
| grade | 범위 기준은 요구사항 미정 |
| gpa | 범위 기준은 요구사항 미정 |
| email | 형식 기준은 요구사항 미정 |
| mobile_phone | 형식 기준은 요구사항 미정 |

## 2. 사용자 정보 검증

| 필드 | 기준 |
|------|------|
| user_id | 빈 값 불가, 중복 불가 |
| password | 빈 값 불가 |
| department | 빈 값 불가 |
| name | 빈 값 불가 |

## 3. 메시지 검증

- `TYPE` 은 `REQ` 또는 `RES`.
- `COMMAND` 는 정의된 명령 코드만 허용.
- 필드는 탭 문자 기준으로 해석한다.
