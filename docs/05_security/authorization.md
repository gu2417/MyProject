# 권한

## 1. 권한 기준

| 기능 | 권한 |
|------|------|
| 로그인 | 연결된 사용자 |
| 학생 검색 | 로그인된 사용자 |
| 학생 추가 | 로그인된 사용자 |
| 학생 삭제 | 로그인된 사용자 |
| 학생 정보 수정 | 로그인된 사용자 |
| 학과명 기반 전체 학생 조회 | 로그인된 사용자 |
| 신규 사용자 추가 | 로그인된 사용자 |
| 사용자 삭제 | 요구사항상 제공, 세부 권한은 로그인 사용자 기준으로 처리 |

## 2. 처리 원칙

서버는 클라이언트 요청 처리 전에 로그인 상태를 확인해야 한다.
