# 클라이언트 컴포넌트

## 1. 클라이언트 계층

| 계층 | 모듈 | 책임 |
|------|------|------|
| Entry | `client_main.c` | 서버 접속, 클라이언트 실행 루프 |
| UI | `client_ui.c` | 메뉴 출력, 사용자 입력 수집 |
| Protocol | `protocol.h` | 메시지 포맷과 명령 코드 참조 |

## 2. 메뉴 범위

| 상태 | 메뉴 |
|------|------|
| 로그인 전 | 로그인 |
| 로그인 후 | 학생 검색, 학생 추가, 학생 삭제, 학생 정보 수정, 학과명 기반 전체 학생 조회, 신규 사용자 추가, 사용자 삭제 |

## 3. 출력 기준

클라이언트는 서버 응답의 `STATUS` 와 `PAYLOAD` 를 해석하여 사용자에게 결과를 출력한다.
